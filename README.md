# Clipla 🎬🤖

**Konuş, Clipla yapsın.**

Clipla, konuşma tabanlı komutlarla videoları otomatik olarak analiz eden, edit planı çıkaran ve render alabilen bir **AI video editing MVP**’sidir.

---

## 🚀 Ne yapar?

1. **Video yükle**
2. **Konuşma / komut ile AI edit planı oluştur**
3. **Otomatik render al ve MP4 indir**

Kullanım senaryosu özellikle:
- YouTube Shorts
- Reels
- TikTok
- Kısa, dikkat çekici sosyal medya videoları

---

## 🧠 MVP Özellikleri

- 🎙️ Konuşma & komut tabanlı edit mantığı  
- ✂️ Sessizlik, tempo ve segment analizi  
- 🧩 Hook / middle / closing ayrımı  
- ⚙️ FastAPI backend
- 🔐 Token bazlı demo erişimi
- ☁️ Cloud deploy (demo amaçlı)
